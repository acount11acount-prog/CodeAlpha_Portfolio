# Personal Portfolio Website

## Overview

This project is a simple and responsive personal portfolio website built using **HTML, CSS, and JavaScript**.
It showcases a developer profile, skills, resume download section, and contact form.

The purpose of this project is to demonstrate basic frontend development skills and create a clean portfolio layout suitable for beginners.

## Features

* Responsive portfolio layout
* Hero section with profile image
* About section
* Skills section
* Resume download button
* Contact form
* Smooth scrolling navigation

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Project Structure

```
portfolio/
│
├── index.html      # Main website structure
├── style.css       # Styling and layout
├── script.js       # Interactive behavior
├── resume.pdf      # Sample resume file
└── profile.jpg     # Profile image used in hero section
```

## Important Note

The resume included in this project is **completely fictional**.

* The name **Ritika Tripathi** is used only as a placeholder.
* All details such as education, experience, and skills inside the resume are **fake and created only for demonstration purposes**.
* The profile image is also a placeholder and **does not represent a real person**.

This project is intended **only for educational and portfolio demonstration purposes**.

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.

No additional installation or dependencies are required.
